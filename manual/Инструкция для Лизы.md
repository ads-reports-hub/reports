# Как работает система отчётов

## Коротко

1-го числа каждого месяца система сама собирает отчёты за прошедший месяц по всем активным клиентам из твоей таблицы. Ты ничего не запускаешь. Готовая ссылка на отчёт появляется прямо в таблице, в строке клиента, в колонке `last_report_link`. Эту ссылку ты просто пересылаешь клиенту в Телеграм.

Каждый отчёт остаётся жить по своей ссылке навсегда, старые месяцы не стираются. Можно в любой момент открыть отчёт за любой прошлый период.

## Таблица - твоя панель управления

Всё управление происходит в одной вкладке: **"Список клиентов на старт"** в твоей гугл-таблице. Колонки:

| Колонка | Что писать |
|---|---|
| КЛИЕНТ: | Название клиента, как оно должно выглядеть в отчёте |
| ТОКЕН | Токен доступа Meta (см. ниже, как его получить) |
| slug | Короткое имя латиницей для ссылки, например `garden-bar`. Маленькими буквами, без пробелов, слова через дефис |
| account_id | ID рекламного кабинета, начинается с `act_` |
| show_dynamics | `да` или `нет`. Показывать ли в отчёте сравнение с прошлым месяцем (стрелочки и проценты). Для новых клиентов лучше `нет`, включишь позже, когда будет что сравнивать |
| active | `да` или `нет`. Пока тут не `да`, система клиента не трогает. Можно заполнять строку постепенно и включить, когда всё готово |
| last_report_link | Сюда система сама пишет ссылку на готовый отчёт. Руками не заполнять |
| last_updated | Дата последнего отчёта, тоже заполняется само |

## Как добавить нового клиента

Два шага. Первый делается в Meta один раз, второй в таблице.

### Шаг 1. Дать доступ к рекламному кабинету (один раз на клиента)

Это ограничение самой Meta, без него токен не будет видеть кабинет.

1. Открой Meta Business Suite, раздел Business Settings (Настройки компании).
2. Слева: Аккаунты → Рекламные аккаунты. Выбери кабинет нового клиента.
3. Нажми "Добавить людей" / Assign People and Assets.
4. Добавь своего системного пользователя (System User, тот же, что использовался для Garden Bar).
5. Дай доступ минимум "Просмотр эффективности" (View performance).
6. Запиши ID кабинета (число вида `act_123456789...`), он понадобится для таблицы.

Если кабинет принадлежит самому клиенту, а у тебя только партнёрский доступ, этот шаг делает админ клиента со своей стороны.

### Шаг 2. Заполнить строку в таблице

Добавь строку, заполни колонки из списка выше, поставь `active` = `да`. Всё. Со следующего 1-го числа по этому клиенту начнут приходить отчёты.

## Если клиент подключился в середине месяца

Ничего специально делать не надо. Отчёт 1-го числа всё равно придёт за календарный месяц, просто данные в нём будут с того дня, когда реально крутилась реклама. Это нормально.

## Если отчёт не пришёл

Самая частая причина: токен перестал работать. Meta иногда сбрасывает токены, например при смене пароля. Что делать:

1. Зайди в Meta Business Settings → Системные пользователи (System Users).
2. Выбери своего системного пользователя, нажми "Создать новый токен" (Generate New Token).
3. Галочки нужны те же, что раньше: `ads_read` и `read_insights`.
4. Скопируй новый токен в колонку ТОКЕН в таблице, старый просто затри.

Отчёт по этому клиенту придёт при следующем запуске. Если нужно раньше или что-то не получается, напиши Максиму.

## Что в отчёте

Отчёт открывается по ссылке на любом телефоне и компьютере. Внутри три переключателя: тёмная/светлая тема, краткая/подробная версия, русский/английский язык. Цифры берутся напрямую из рекламного кабинета Meta, тексты выводов и рекомендаций пишутся автоматически в твоём стиле. Перед отправкой клиенту стоит пробежаться глазами: цифры всегда точные, но формулировки при желании можно попросить Максима подправить.
