# docs/

Это корень, который реально отдаёт GitHub Pages (`/docs` на ветке `main`).
Файлы сюда пишет только `engine/orchestrator.py` во время автоматического
прогона — руками здесь ничего не редактировать.

Структура: `docs/<слаг клиента>/<YYYY-MM>/index.html` + `assets/`. Старые
периоды никогда не удаляются и не перезаписываются.
